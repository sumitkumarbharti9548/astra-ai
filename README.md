# 🚀 Astra AI

Astra AI is a **student-focused frontend web platform** designed to provide a modern interface for learning, coding practice, and accessing study resources.

> ⚠️ Note: This is currently a **static website (HTML, CSS, JavaScript only)**. Backend features are planned for future updates.

---

## 🌟 Current Features (Frontend)

* 🎨 Modern and responsive UI design
* 📚 Study material pages (UI)
* 💻 Coding practice section (UI)
* 🤖 AI chatbot interface (UI only)
* 📝 Resume builder interface
* 📂 Notes upload page (UI)
* 📖 Notes generator interface
* 📞 Contact Us page
* ℹ️ About Astra AI page

---

## 🚧 Upcoming Features (Planned)

* 🔐 User Authentication (Login/Signup)
* 🗄️ Database integration (Firebase / MongoDB)
* 📂 Real Notes Upload & Download system
* 🤖 Working AI Chatbot integration
* 💻 Live Coding Playground
* 🏆 Daily Coding Challenges system

---

## 🎯 Objective

The goal of Astra AI is to build an **all-in-one platform for students** where they can:

* Learn new concepts
* Practice coding
* Build resumes
* Access study materials
* Solve doubts using AI
